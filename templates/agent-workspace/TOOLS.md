# TOOLS.md - 本地工具配置

技能定义工具*如何*工作，本文件记录*你的*具体配置——属于你独有的设置。

## 应该记录什么

例如：
- 首选数据源
- API 端点
- 自定义指标
- 策略模板
- 任何环境特定配置

## 示例

```markdown
### 数据源
- Binance API - 主要市场数据
- CoinGecko - 价格聚合

### 自定义指标
- RSI_EMA_Combo - RSI + EMA 交叉信号

### 策略模板
- 网格交易 - 多层级买卖单
- 定投策略 - 定期定额投资设置
```

## 为什么要分开？

技能是共享的，你的配置是私有的。分开存放意味着你可以更新技能而不丢失配置，也可以分享技能而不泄露基础设施信息。
