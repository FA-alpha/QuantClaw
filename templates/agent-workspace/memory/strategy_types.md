# 策略类型分类规则

## 核心分类

| 类型 | 识别方法 | 说明 |
|------|---------|------|
| **马丁策略** | 名称含"风霆" | 风霆现货、风霆V4等 |
| **网格策略** | strategy_type=7 | 唯一网格策略 |
| **趋势策略** | 其他所有 | 鲲鹏系列等 |

---

## 数据获取

**策略类型列表已自动缓存**
- 首次查询时自动获取
- 无需手动执行命令
- 查询脚本会自动使用缓存

---

## 使用示例

**用户说"马丁策略"**：
- 查询缓存中名称含"风霆"的策略
- 提取 strategy_type ID（如 `1, 11`）
- 使用 `--strategy-types "1,11"` 查询

**用户说"网格策略"**：
- 直接使用 `--strategy-types "7"`

**用户说"趋势策略"**：
- 排除马丁(1,11)和网格(7)
- 使用其他所有 strategy_type ID
